# Invoice Generator

A professional, easy-to-use invoice generator with real-time preview and customizable templates.

**[🚀 Live Demo](https://luxe-ai.github.io/invoice-generator/)**

## Features

- **Real-time Preview**: See your invoice update as you type
- **Customizable Items**: Add/remove items with ease
- **Multiple Currencies**: Support for USD, CNY, HKD, EUR, GBP
- **Bank Information**: Editable payment details
- **Template System**: Save and load invoice templates
- **Export Options**: Download as HTML or print to PDF
- **Signature Section**: Professional signature area
- **Responsive Design**: Works on desktop and mobile

## Usage

Simply open `outputs/invoice-generator.html` in your browser to start creating invoices.

### Quick Start

1. Fill in your information (sender name, company)
2. Enter client details (bill to)
3. Add invoice items with descriptions and amounts
4. Customize bank information for payment
5. Export or print your invoice

## Features in Detail

### Invoice Details
- Invoice number
- Date
- Currency selection

### Sender Information
- Your name
- Company name (optional)

### Client Information
- Client name
- Client company

### Items
- Description
- Amount (with automatic formatting)
- Add/remove multiple items

### Bank Information
- Bank name
- Account name
- Account number

### Export Options
- **Download HTML**: Get a standalone HTML file
- **Print/PDF**: Print directly or save as PDF
- **Save Template**: Store your configuration for future use
- **Load Template**: Quickly recall saved settings

## Technology

Built with vanilla HTML, CSS, and JavaScript - no dependencies required!
